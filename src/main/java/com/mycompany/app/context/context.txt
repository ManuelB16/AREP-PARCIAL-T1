Descripción:
Usted debe construir una "Calculadora Web para estimar la media y la desviación estándar de un conjunto de números ". La solución consta de un servidor backend que responde a solicitudes HTTP GET de la Facade, un servidor Facade que responde a solicitudes HTTP GET del cliente , y un cliente Html+JS que envía los comandos y muestra las respuestas. El cliente HTML+Js hace llamadas asíncronas a la fachada. La api permite almacenar números reales, borrar la lista de números y calcular la media y la desviación estándar. 
 
API mínimo (solo GET) para el escenario con Fachada y Backend, cubre las funciones de agregar un número real a la lista de números, borrar toda la lista, mostrar la lista y calcular la media y la desviación estándar.

Debe usar sockets solamente, no puede usar ni Spark ni Spring.
 
Descripción
Desarrolle una solución compuesta por:

Servidor Backend: mantiene en memoria una lista de números reales y expone un API solo GET sobre sockets (sin frameworks).
Servidor Fachada: expone el mismo API solo GET y reenvía las solicitudes al Backend (actúa como proxy/gateway), también sobre sockets.
Cliente HTML+JS: realiza peticiones GET a la Fachada y muestra las respuestas.
Restricciones
Usar sockets (p. ej., ServerSocket en Java). No usar frameworks (ni Spark ni Spring ni similares).
El API es solo GET.
Las respuestas deben ser JSON con Content-Type: application/json.
La desviación estándar debe ser la poblacional (dividir por n).
El cliente HTML+Js hace llamadas asíncronas a la fachada
